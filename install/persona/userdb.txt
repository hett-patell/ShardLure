# Realistic weak creds. ASCII ONLY: cowrie reads userdb.txt as strict ASCII
# (auth.py: read_text(encoding="ascii")), so a single non-ASCII byte -- e.g. a
# unicode em-dash in a comment -- makes the whole userdb load throw and cowrie
# silently reverts to built-in defaults, so NONE of these creds apply. Keep
# every line 7-bit ASCII. Avoid a '*:*' catch-all (accept-everything is a
# honeypot tell that fingerprinting bots probe for).
#
# Format: user:x:password   '!'=deny   '*'=any   '/regex/'=match. First match
# wins. Passwords below are the ones real botnets most frequently brute-force
# (mined from captured cowrie login attempts), so sessions reach post-auth
# where the useful telemetry (commands, payloads, tty replays) is.

# Deny obvious honeypot-detection probe strings first.
*:x:!/honeypot|cowrie|thisisnotarealpassword/i
